## 1. Généralités
Cette application sert de télécommande pour les téléviseurs Philips sur le réseau local.

Le développeur de cette application ne collecte, ne stocke ni ne traite aucune donnée personnelle.

## 2. Communication réseau
L'application communique exclusivement avec le téléviseur Philips configuré par l'utilisateur.

Seules les commandes techniques nécessaires au contrôle du téléviseur sont transmises (p. ex. changement de chaîne, volume, navigation).

Cette communication s'effectue directement entre votre appareil et le téléviseur. Aucune donnée n'est transmise à des serveurs externes.

## 3. Stockage des données
L'application ne stocke aucune donnée personnelle.

Les paramètres techniques suivants sont stockés localement sur votre appareil :

- Adresse IP du téléviseur Philips
- Certificat d'appairage pour l'authentification

Ces données restent exclusivement sur votre appareil.

## 4. Autorisations
L'application nécessite un accès réseau pour communiquer avec le téléviseur Philips sur le réseau local.

Aucune autre autorisation sensible n'est utilisée.

## 5. Tiers
Cette application n'utilise aucun service tiers, outil d'analyse ni publicité.

## 6. Modifications de cette politique de confidentialité
Cette politique de confidentialité peut être mise à jour si nécessaire pour refléter les modifications de l'application ou les exigences légales.
