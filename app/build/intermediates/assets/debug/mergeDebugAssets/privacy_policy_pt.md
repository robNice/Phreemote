## 1. Geral
Este aplicativo funciona como controle remoto para televisores Philips na rede local.

O desenvolvedor deste aplicativo não coleta, armazena nem processa nenhum dado pessoal.

## 2. Comunicação de rede
O aplicativo se comunica exclusivamente com o televisor Philips configurado pelo usuário.

Apenas os comandos técnicos necessários para o controle do televisor são transmitidos (ex.: mudança de canal, volume, navegação).

A comunicação ocorre diretamente entre o seu dispositivo e o televisor. Nenhum dado é transmitido a servidores externos.

## 3. Armazenamento de dados
O aplicativo não armazena nenhum dado pessoal.

As seguintes configurações técnicas são armazenadas localmente no seu dispositivo:

- Endereço IP do televisor Philips
- Certificado de emparelhamento para autenticação

Esses dados permanecem exclusivamente no seu dispositivo.

## 4. Permissões
O aplicativo requer acesso à rede para comunicar-se com o televisor Philips na rede local.

Nenhuma outra permissão sensível é utilizada.

## 5. Terceiros
Este aplicativo não utiliza serviços de terceiros, ferramentas de análise nem publicidade.

## 6. Alterações nesta política de privacidade
Esta política de privacidade pode ser atualizada conforme necessário para refletir alterações no aplicativo ou requisitos legais.
