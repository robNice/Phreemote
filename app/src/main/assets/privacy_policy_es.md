## 1. General
Esta aplicación funciona como mando a distancia para televisores Philips en la red local.

El desarrollador de esta aplicación no recopila, almacena ni procesa ningún dato personal.

## 2. Comunicación de red
La aplicación se comunica exclusivamente con el televisor Philips configurado por el usuario.

Solo se transmiten los comandos técnicos necesarios para el control del televisor (p. ej. cambio de canal, volumen, navegación).

La comunicación se realiza directamente entre su dispositivo y el televisor. No se transmiten datos a servidores externos.

## 3. Almacenamiento de datos
La aplicación no almacena ningún dato personal.

Los siguientes ajustes técnicos se almacenan localmente en su dispositivo:

- Dirección IP del televisor Philips
- Certificado de emparejamiento para la autenticación

Estos datos permanecen exclusivamente en su dispositivo.

## 4. Permisos
La aplicación requiere acceso a la red para comunicarse con el televisor Philips en la red local.

No se utilizan otros permisos sensibles.

## 5. Terceros
Esta aplicación no utiliza servicios de terceros, herramientas de análisis ni publicidad.

## 6. Cambios en esta política de privacidad
Esta política de privacidad puede actualizarse si es necesario para reflejar cambios en la aplicación o requisitos legales.
