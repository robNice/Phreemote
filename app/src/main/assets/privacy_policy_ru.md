## 1. Общие положения
Это приложение выполняет функцию пульта дистанционного управления для телевизоров Philips в локальной сети.

Разработчик этого приложения не собирает, не хранит и не обрабатывает никаких персональных данных.

## 2. Сетевое взаимодействие
Приложение взаимодействует исключительно с телевизором Philips, настроенным пользователем.

Передаются только технические команды, необходимые для управления телевизором (например, переключение каналов, громкость, навигация).

Обмен данными происходит напрямую между вашим устройством и телевизором. Данные не передаются на внешние серверы.

## 3. Хранение данных
Приложение не хранит персональные данные.

На вашем устройстве локально сохраняются следующие технические настройки:

- IP-адрес телевизора Philips
- Сертификат сопряжения для аутентификации

Эти данные хранятся исключительно на вашем устройстве.

## 4. Разрешения
Приложение требует доступа к сети для взаимодействия с телевизором Philips в локальной сети.

Другие конфиденциальные разрешения не используются.

## 5. Третьи стороны
Приложение не использует сторонние сервисы, аналитические инструменты или рекламу.

## 6. Изменения в политике конфиденциальности
Настоящая политика конфиденциальности может обновляться по мере необходимости для отражения изменений в приложении или требований законодательства.
